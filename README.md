## How to run the app

Follow these steps to set up and start the project:

1.  **Clone the repository**:
    ```bash
    git clone
    ```
2.  **Install dependencies**:
    Make sure you have [Node.js](https://nodejs.org) installed. Run:
    ```bash
    npm install
    ```
3.  **Start the development server**:
    ```bash
    npm run dev
    ```
4.  **Open the app**:
    Once the server is running, open the URL provided in your terminal (usually `http://localhost:5173` or similar).

## Project Structure

Below is an overview of the project's folder structure and the types of files they contain:

- **src/** — The main source directory:
  - **assets/** — Contains static vector graphics and icons (`.svg` files).
  - **main.js** — The core JavaScript file containing the application logic, API interaction, and DOM updates.
  - **style.css** — The main stylesheet containing CSS variables, layouts, and theme styles.
- **index.html** — The main HTML entry point of the application.
